#### 🎮 [Portal Game - Landing Page Imersiva](https://github.com/guilhermeoy1987/modulo5sitedegames)
*Data: 2026 | Deploy:* [Acessar Site](https://projeto3sitegame.vercel.app)

Desenvolvimento de uma Landing Page temática para o setor de games, focada em fidelidade visual e técnicas avançadas de estilização e automação.

* ✅ **Estilização com SASS:** Uso de variáveis, mixins e aninhamento para um código CSS mais limpo, modular e fácil de manter.
* ✅ **Automação de Workflow:** Implementação de automatizadores (Gulp/Grunt) para compilação de arquivos e otimização de assets.
* ✅ **Experiência do Usuário (UX):** Design responsivo com transições suaves e layout planejado para alta conversão e engajamento.
* ✅ **Hospedagem:** Deploy contínuo via **Vercel**, garantindo performance e carregamento rápido.

**Stack:** HTML5, SASS, JavaScript, Gulp/Grunt e Vercel.
